# switch 语句
```
switch(n)
{
case 1:
  执行代码块 1
  break;
case 2:
  执行代码块 2
  break;
default:
  n 与 case 1 和 case 2 不同时执行的代码
}
```

# for 循环
for 循环是您在希望创建循环时常会用到的工具。
```javascript
for (语句 1; 语句 2; 语句 3)
  {
  被执行的代码块
  }
  
  /*
    语句 1 在循环（代码块）开始前执行

    语句 2 定义运行循环（代码块）的条件

    语句 3 在循环（代码块）已被执行之后执行
  */
```

```javascript
for(表达式1 （变量声明）; 表达式2 （条件） ;  表达式3 （自增或自减）){
    代码块
}

当for循环语句被解析时候

开始（初始化）  表达式1执行（执行并且只有一次）

循环判断  表达式2的结果为真 =>  执行代码块  =>执行表达式3  => 循环判断  
               结果为假 =>  结束循环
```
