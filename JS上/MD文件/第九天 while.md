# while 循环  
```
while(表达式（一般是条件 不能省略）){
    循环体
    [增量]
}
```
循环判断 =》表达式的结果转布尔值为真  =》执行循环体  =》循环判断
         =》表达式的结果转布尔值为假  =》while循环语句结束（跳出while循环）
--- 
break 跳出while循环
